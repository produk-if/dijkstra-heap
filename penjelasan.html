üíª Kode Program

generate_fibonacci.c 
// Global array untuk menyimpan deret Fibonacci
long long fib[50];

void generate_fibonacci(int n) {
    fib[0] = 0;    // F(0) = 0
    fib[1] = 1;    // F(1) = 1
    
    for (int i = 2; i < n; i++) {
        fib[i] = fib[i-1] + fib[i-2];  // RUMUS FIBONACCI!
    }
}

Algoritma Fibonacci Search

üí°
Apa itu Fibonacci Search?
Algoritma pencarian pada array terurut yang menggunakan deret Fibonacci untuk menentukan posisi pembagian (bukan bagi 2 seperti Binary Search).

üÜö Perbandingan dengan Binary Search

Aspek Binary Search Fibonacci Search
Pembagian Array 50% - 50% ~62% - 38% (Golden Ratio)
Operasi Matematika Perlu division (/) Hanya + dan -
Kompleksitas O(log n) O(log n)

üìã Langkah-Langkah Pencarian:

1 Cari F(k) ‚â• 11: F(7) = 13 ‚â• 11 ‚úì

2 index = -1 + 5 = 4: arr[4] = 10

3 index = 4 + 3 = 7: arr[7] = 16
16 > 14 ‚Üí geser KIRI
4 index = 4 + 1 = 5: arr[5] = 12
12 < 14 ‚Üí geser KANAN, offset = 5
5 index = 5 + 1 = 6: arr[6] = 14
14 == 14 ‚Üí KETEMU di index 6! ‚úì

üíª Kode Program

fibonacci_search.c 
int fibonacci_search(int arr[], int n, int target) {
    // Cari F(k) >= n
    int k = 0;
    while (fib[k] < n) k++;
    
    int offset = -1;
    
    while (k > 1) {
        int i = offset + fib[k-2];
        if (i >= n) i = n - 1;
        
        if (arr[i] < target) {
            // Target di KANAN
            k = k - 1;
            offset = i;
        }
        else if (arr[i] > target) {
            // Target di KIRI
            k = k - 2;
        }
        else {
            // KETEMU!
            return i;
        }
    }
    
    return -1;  // Tidak ditemukan
}

Parallelisasi dengan OpenMP

‚ö°
OpenMP menggunakan #pragma directive untuk membagi pekerjaan ke beberapa thread secara otomatis.

üîÑ Serial vs Parallel

‚ùå Serial (Lambat)

Pencarian dilakukan satu per satu

Search 1 ‚Üí Search 2 ‚Üí Search 3 ‚Üí ...

‚úÖ OpenMP (Cepat)

Pencarian dilakukan bersamaan

Thread 1: Search 1, 5, 9...
Thread 2: Search 2, 6, 10...
Thread 3: Search 3, 7, 11...
Thread 4: Search 4, 8, 12...

üíª Kode Program

batch_search_openmp.c 
void batch_search_openmp(int arr[], int n, 
                         int targets[], int num_targets, 
                         int results[]) {
    
    #pragma omp parallel for
    for (int i = 0; i < num_targets; i++) {
        results[i] = fibonacci_search(arr, n, targets[i]);
    }
}

/*
 * #pragma omp parallel for:
 * - Membagi loop ke beberapa thread
 * - Setiap thread cari target berbeda
 * - Tidak perlu sinkronisasi (hasil di index berbeda)
 */

Parallelisasi dengan Cilk

üîÄ
Cilk menggunakan model task-based parallelism dengan konsep spawn (buat task) dan sync (tunggu selesai).

üå≥ Fibonacci Rekursif - Contoh Klasik Cilk

KONSEP CILK
F(n) = F(n-1) + F(n-2)
= [TASK 1] + [TASK 2]
Kedua task INDEPENDEN, bisa dijalankan PARALEL!

üå≤ Visualisasi Tree F(5)

                    F(5)
                   /    \
              F(4)  +   F(3)        ‚Üê 2 task paralel
              / \        / \
          F(3)+F(2)  F(2)+F(1)      ‚Üê 4 task paralel
          / \    |    |    |
      F(2)+F(1) ...  ...   ...       ‚Üê 8 task paralel

üíª Kode Program - Fibonacci Cilk

fibonacci_cilk.c 
long long fibonacci_cilk(int n) {
    if (n <= 1) return n;
    
    long long x, y;
    
    #pragma omp task shared(x)
    x = fibonacci_cilk(n-1);     // SPAWN task 1
    
    #pragma omp task shared(y)
    y = fibonacci_cilk(n-2);     // SPAWN task 2
    
    #pragma omp taskwait         // SYNC - tunggu selesai
    
    return x + y;                // Gabungkan hasil
}

üíª Kode Program - Batch Search Cilk

batch_search_cilk.c 
void batch_search_cilk(int arr[], int n, 
                       int targets[], int start, int end, 
                       int results[]) {
    int count = end - start;
    
    // Base case: sequential jika batch kecil
    if (count <= THRESHOLD) {
        for (int i = start; i < end; i++) {
            results[i] = fibonacci_search(arr, n, targets[i]);
        }
        return;
    }
    
    // Divide: bagi menjadi 2 bagian
    int mid = start + count / 2;
    
    #pragma omp task
    batch_search_cilk(arr, n, targets, start, mid, results);
    
    #pragma omp task
    batch_search_cilk(arr, n, targets, mid, end, results);
    
    #pragma omp taskwait  // Sync
}
